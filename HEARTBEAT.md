HEARTBEAT.md

This repository uses a Paperclip/agent-driven workflow. Each heartbeat the active
agent should read this file. It contains minimal guidance for using the Paperclip
API and running short maintenance tasks.

- Read this file at the start of every heartbeat.
- Use the `gh` CLI for PR operations when asked to create PRs.
- When creating commits programmatically, keep commits small and describe the why.

If you are unsure how to proceed, ask for clarification in the issue thread.
